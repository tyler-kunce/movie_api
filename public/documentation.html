Documentation: Great Sports Movies

This web application will provide users with information about different sports movies as well as their respective actors, directors, and genres. Users will be able to sign up, update their personal information, and create a list of favorite movies from the database.

Business Logic URL Endpoint HTTP Method Request Format Response Format
Get a list of all movies /movies GET None A JSON object holding data about all movies
Get data about a single movie /movies/[Title] GET None A JSON object holding data about a single movie, containing a title, director, and lead actor. Example: { title: 'Without Limits', director: 'Robert Towne', actorLead: 'Billy Crudup' }
Get data about a genre, by title /movies/genres/[Name] GET None A JSON object holding data about a genre, by genre name. Example: { Name: 'Biographical', Description: 'Biographical, otherwise called a biopic...' }
Get data about a director, by name /movies/directors/[Name] GET None A JSON object holding data about a director, by director name. Example: { Name: 'Robert Towne', Bio: 'An American screenwriter and director.', }
Add data about a new user /users POST A JSON object holding data about the user to add A JSON object holding data about the user that was added, including an ID
Update data about an existing user /users/[Username] PUT None A text message indicating the data that was updated
Add movie to user's favorite's list /users/[Username]/movies/[MovieID] POST A JSON object holding data about the movie that was added A JSON object holding data about the movie that was added
Delete movie from a user's favorite's list /users/[Username]/movie/[MovieID] DELETE None A text message indicating whether the movie was successfully removed
Delete a user /users/[Username] DELETE None A text message indicating whether the user was successfully removed
